# Box Component

Компонент `Box` предоставляет универсальный контейнер, который можно стилизовать и настраивать с помощью различных пропсов. Это гибкий компонент для создания макетов и компоновки элементов.

## Пропсы

У `Box` на данный момент реализованы следующие свойства `height` `width` `as`

## Использование

Компонент `Box` может быть использован для создания блоков с заданными шириной и высотой. Вы также можете изменить используемый HTML-элемент, передав проп `as`.

### Базовый пример как `div`

```jsx
<Box width={200} height={100}>
  Это контент внутри Box
</Box>

### Box как section

<Box as="section" width="50%" height="30vh">
  Это секция
</Box>

### Box как section

<Box as="aside" width={300} height={150}>
  Это боковая панель (aside)
</Box>
